# Digital Banking Funnel & Transaction Analytics

## Overview
This project analyzes a digital banking platform to understand customer onboarding, transaction success rates, and product usage trends.

## Problem Statement
A banking application wants to identify:
- Where users drop off during onboarding
- Which channels bring high-quality users
- Why transactions fail

## Dataset
The project uses three datasets:
- Customers data
- Customer onboarding funnel data
- Transactions data

## Tools Used
- SQL
- Python
- Power BI / Tableau

## Key Analysis
- Funnel conversion rates
- Monthly active users
- Transaction success vs failure rates
- Channel performance analysis

## Goal
To provide data-driven insights that improve customer onboarding, reduce transaction failures, and enhance product performance.
